The component class, e.g. PaperSlider.java you get when using the component starter, see Integrating a Web Component is just a stub which handles the imports. There are multiple ways to interact with a web component but the typical pattern is: Use properties on the element to define how it should behave Listen to events on the element to get notified of when the user does something Call functions on the element to perform specific tasks such as open a popup Add sub elements to define child contents core-hello Meatspace!
