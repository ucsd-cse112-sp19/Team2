    meat-input

    displays a stylized input field

    <meat-input autocomplete bootstrap="[string]" disabled limit="[integer]" password placeholder="[string]" readonly size="[string]" suggest type="[string]" value="[string]">
    Properties:
    Name Type Description
    autocomplete attribute

    Enables autocomplete by caching previous input.
    bootstrap string

    Enables bootstrap as styling of the button.
    disabled attribute

    Disables input from accepting events.
    limit integer

    Restricts how many characters can be entered in the input.
    password attribute

    Format input as a password field.
    placeholder string

    Describes what goes into the input initially.
    readonly attribute

    Disables input from being written.
    size string

    Determines the size of the input box.
    suggest attribute

    Enables the dropdown of suggestions for the input.
    type string

    Describes the type of input.
    value string

    The contents of the input.
