    meat-button

    meat-button webcomponent

    displays a stylized button

    <meat-button autofocus="[boolean]" circle="[string]" disabled="[boolean]" round="[boolean]" size="[string]" type="[string]">
    Properties:
    Name Type Description
    autofocus boolean

    Enables button to automatically get focus when the page loads.
    circle string

    Enables a circle shaped button.
    color string

    Sets the color of the button.
    disabled boolean

    Disables button from accepting events.
    round boolean

    Enables a round shaped button.
    size string

    Changes the size of the button.
    type string

    Enables the specified type of button.
