    meat-input

    meat-input webcomponent

    displays an input box

    <meat-input disabled="[boolean]" size="[string]" limit="[string]" placeholder="[string]" password="[boolean]" value="[string]" readonly="[boolean]" suggest="[on, off]" autocomplete="[on, off]">
    Properties:
    Name Type Description
    disabled boolean

    Disables input from accepting text.
    size string

    Changes the size of the input box.
    placeholder string

    Placeholder showing what text this input can take.
    password boolean

    Make the input a password field and replace text with asteriks.
    value string

    Default text value within the input box.
    readonly boolean

    Determines if the input box can be edited.
    suggest on, off

    Determines if input box will display a predetermined suggestion list.
    autocomplete on, off

    Determines if input box will use native browser capabilit to display user's previous inputs as suggestion.
